# 🎮 KBC Quiz Game

A "Kaun Banega Crorepati" inspired quiz game built with Python.

## 🛠️ Technologies Used
- Python
- (Tkinter / any library you used)

## 🎯 Features
- Multiple choice questions (A, B, C, D)
- Lifelines: 50-50, Phone a Friend, Audience Poll
- Prize money progression

## 🚀 How to Run
1. Clone the repository
```bash
   git clone https://github.com/moinshaikh03/KBC-Quiz-Game.git
```
2. Navigate to the folder
```bash
   cd KBC-Quiz-Game
```
3. Run the game
```bash
   python main.py
```
